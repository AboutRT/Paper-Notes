# Prompt-Free Universal Region Proposal Network

**会议**: CVPR 2026  
**arXiv**: [2603.17554](https://arxiv.org/abs/2603.17554)  
**代码**: [GitHub](https://github.com/tangqh03/PF-RPN)  
**领域**: 目标检测  
**关键词**: 区域提案, 无提示检测, 零样本泛化, 可学习嵌入, 开放世界

## 一句话总结

无需文本/图像提示的通用区域提案网络，用5% COCO数据训练零样本泛化到19个跨域数据集

## 研究背景与动机

现有RPN依赖类别名/示例图像，无法适应未知目标类别场景

## 方法详解

### 整体框架

SIA初步定位→CSP级联自提示捕获难例→CG-QS中心性引导选择高质量查询

### 关键设计

1. SIA(稀疏图像感知适配器)：可学习嵌入+多层级视觉特征聚合
2. CSP(级联自提示)：迭代精化捕获小/遮挡目标
3. CG-QS(中心性引导)：中心性评分选择高质量查询

### 损失函数 / 训练策略

多目标联合优化，平衡各组件贡献。详见各模块描述。

## 实验关键数据

### 主实验与关键发现

CD-FSOD AR100/300/900提升+6.0/+7.5/+6.6，19个跨域数据集一致有效

## 亮点与洞察

真正无提示检测；数据效率极高(5% COCO)；19个跨域验证

## 局限性 / 可改进方向

仅生成提案不分类；极端域差距泛化上限待探索

## 相关工作与启发

在现有方法基础上的重要改进，所提技术路线可推广到相关任务。
