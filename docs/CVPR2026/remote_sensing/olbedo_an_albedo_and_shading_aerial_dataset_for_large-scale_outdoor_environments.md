# Olbedo: An Albedo and Shading Aerial Dataset for Large-Scale Outdoor Environments

**会议**: CVPR 2026
**arXiv**: [2602.22025](https://arxiv.org/abs/2602.22025)
**代码**: [Project](https://gdaosu.github.io/olbedo/)
**领域**: 遥感
**关键词**: 内在图像分解, 反照率, 航拍数据集, 逆渲染, 城市数字孪生

## 一句话总结

首个大规模真实航拍反照率-着色分解数据集（5664张UAV图像），通过物理逆渲染管线生成真值

## 研究背景与动机

IID分离图像为反照率和着色，是重光照/材质编辑基础。现有数据集要么合成要么室内，室外航拍完全空白

## 方法详解

### 整体框架

RAW格式UAV采集→摄影测量重建→HDR天空辐射采集→逆渲染优化→反照率/着色图+深度+法线

### 关键设计

1. 5664张图像4种地貌跨年份多光照
2. 多视图一致反照率和着色图
3. HDR全天空辐射图
4. 扩散IID模型LoRA微调后室外泛化显著提升

### 损失函数 / 训练策略

多目标联合优化，平衡各组件贡献。详见各模块描述。

## 实验关键数据

### 主实验与关键发现

合成预训练+Olbedo微调显著提升室外反照率预测；支持重光照/材质编辑/场景变化分析

## 亮点与洞察

填补重要空白：首个真实航拍IID数据集；端到端物理管线

## 局限性 / 可改进方向

逆渲染真值不如合成精确；仅4个场景；Lambertian假设

## 相关工作与启发

在现有方法基础上的重要改进，所提技术路线可推广到相关任务。
