# S2M-Former: Spiking Symmetric Mixing Branchformer for Brain Auditory Attention Detection

**会议**: NeurIPS 2025  
**arXiv**: [2508.05164](https://arxiv.org/abs/2508.05164)  
**代码**: https://github.com/JackieWang9811/S2M-Former  
**领域**: 模型压缩  
**关键词**: 脉冲神经网络, 脑机接口, 听觉注意力检测, 对称双分支, 低功耗

## 一句话总结
提出 S2M-Former，一种基于脉冲神经网络的对称双分支框架，通过通道参数化 LIF 神经元和 1D Token 化设计，在参数减少 14.7 倍、功耗降低 5.8 倍的同时实现与 SOTA 相当的 EEG 听觉注意力检测性能。

## 研究背景与动机
1. **领域现状**：听觉注意力检测（AAD）通过 EEG 信号解码听者在多人对话中的注意焦点，对开发神经操控助听器至关重要
2. **现有痛点**：现有双分支网络（DBPNet 等）虽性能好但参数多（0.88M）、功耗高（0.45mJ），且采用简单拼接融合；3D 卷积建模 EEG 拓扑导致部署困难
3. **核心 idea**：用脉冲神经网络替代 MAC 操作降低功耗，用 1D Token 替代 3D 操作降低参数，对称设计促进互补学习

## 方法详解

### 关键设计
1. **通道参数化 LIF（CPLIF）**：为每个通道分配独立膜时间常数 $\tau_l[c]$，适应 EEG 不同频率成分的时间动态
2. **空间分支编码器（SBE）**：级联时间卷积（1×8, 1×16）+ 双路空间卷积（C×1），分离式时空建模
3. **频率分支编码器（FBE）**：三层卷积块（D→4D→2D→D）+ 空洞卷积（膨胀率=2），捕捉脑地形图的频率空间特征
4. **脉冲门控通道混合（SGCM）**：跨分支融合，通道注意力向量自适应分配权重，仅 6K 参数
5. **膜势感知 Token 混合（MPTM）**：渐进式精化融合，三阶段（GAP 聚合→混合引导→脉冲调制+残差）

## 实验关键数据

### 主实验
| 数据集/模型 | 参数(M) | 2s准确率 | 能耗(mJ) |
|------------|---------|---------|---------|
| DBPNet (ANN) | 0.88 | 93.66% | 0.4526 |
| S2M-Former (SNN) | **0.06** | 93.71% | **0.0779** |

### 消融实验
| 配置 | DTU 2s准确率 |
|------|------------|
| 完整模型 | 85.28% |
| ANN版本 | 80.94% (-4.34%) |
| 仅空间分支 | 82.28% (-3.0%) |
| 仅频率分支 | 70.48% (-14.8%) |

## 亮点与洞察
- 脉冲计算天然适合 EEG 的稀疏变化模式，SNN vs ANN 提升 4.34%
- 1D Token 化将参数从 0.88M 减至 0.06M 而不损性能，实用工程价值高

## 局限性
- 极短窗口（0.1s）下与 SOTA 差距扩大；跨受试者泛化仍有挑战
