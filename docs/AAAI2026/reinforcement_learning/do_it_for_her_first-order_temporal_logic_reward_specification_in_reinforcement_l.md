# Do It for HER: First-Order Temporal Logic Reward Specification in RL

**会议**: AAAI 2026  
**arXiv**: [2602.06227](https://arxiv.org/abs/2602.06227)  
**代码**: [GitHub](https://bit.ly/4i20C4Z)  
**领域**: 强化学习  
**关键词**: 时序逻辑, 奖励规范, LTLfMT, HER, 稀疏奖励

## 一句话总结
提出基于 LTLfMT（线性时序逻辑模理论）的 RL 奖励规范框架，用一阶逻辑谓词替代布尔原子使得奖励可直接定义在连续/异构状态上（无需手工标签函数），并组合 CRM+HER 解决逻辑规范的稀疏奖励问题。

## 研究背景与动机

1. **现有痛点**：LTLf 仅支持布尔谓词，从连续状态到布尔的映射需要手工编码；逻辑奖励天然稀疏。

2. **切入角度**：LTLfMT 将一阶理论（如 NRA）嵌入时序逻辑谓词，用 SMT 求解器替代手工标签函数；用一阶常量的参数化使 HER 自然适配。

## 方法详解

### 整体框架
LTLfMT 公式 $\varphi$ → 命题化为 LTLf $\varphi'$ → DFA → 乘积 MDP $M'$ → DDPG + CRM-HER。

### 关键设计
1. **LTLfMT 可判定片段**：去掉 lookahead 算子，保留时序算子和一阶谓词
2. **SMT 作为通用标签函数**：每步调用 SMT 求解器评估谓词真值
3. **CRM-HER 组合**：CRM 处理 DFA 复杂性，HER 处理目标发现，两者协同

## 实验关键数据
CRM-HER 在最复杂任务 parking_2 上是唯一能学到成功策略的方法。

## 亮点与洞察
- SMT 替代手工标签函数是关键工程创新
- CRM 和 HER 的非平凡组合利用了一阶常量的参数化结构

## 局限性
- SMT 求解器每步调用有性能开销，仅在简单环境验证
