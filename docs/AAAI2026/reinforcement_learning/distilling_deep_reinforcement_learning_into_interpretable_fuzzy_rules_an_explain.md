# Distilling Deep RL into Interpretable Fuzzy Rules

**会议**: AAAI 2026  
**arXiv**: [2603.13257](https://arxiv.org/abs/2603.13257)  
**代码**: 无  
**领域**: 强化学习  
**关键词**: 可解释AI, 模糊逻辑, 策略蒸馏, TSK系统, 连续控制

## 一句话总结
提出层次化 TSK 模糊分类系统，将深度 RL 的神经策略蒸馏为人类可读的 IF-THEN 规则，通过 K-Means 分区状态空间 + Ridge 回归学习局部动作推断，在 LunarLander 上达到 81.5% 保真度（比决策树高 21%），并引入 FRAD/FSC/ASG 三个可量化可解释性指标。

## 研究背景与动机

1. **现有痛点**：SHAP/LIME 只提供局部解释，决策树在连续控制中保真度低，符号回归可扩展性差。

2. **切入角度**：TSK 模糊系统天然桥接数值计算和语言推理，通过层次化设计解决可扩展性。

## 方法详解

### 整体框架
Level 1：K-Means 分区状态空间（前件学习）→ Level 2：每个区域 Ridge 回归学 TSK 后件 $f_i(s) = w_i^T s + b_i$。

### 关键设计
- **三角 vs 高斯隶属函数**：三角函数产生更稀疏的规则激活→更聚焦的解释
- **三个可解释性指标**：FRAD（规则激活密度）、FSC（模糊集覆盖）、ASG（动作空间粒度）
- **DTW 验证行为保真度**：比 MSE 更适合序列决策

## 实验关键数据

| 模型 | 保真度(%) | FRAD |
|------|---------|------|
| Decision Tree-16 | 60.14 | N/A |
| FCS-Gaussian-16 | 81.38 | 0.723 |
| **FCS-Triangular-16** | **81.48** | **0.814** |

## 亮点与洞察
- 三个可量化指标填补 XRL 领域空白
- 仅 4 条规则就能达到高保真度——复杂神经策略可蒸馏为极简结构

- 仅在 LunarLander 一个环境验证
